# LSPBEA Indonesia Website

## Overview
A static website that replicates the design and layout of the Building Engineers Association website, with all references to "BUILDING ENGINEERS ASSOCIATION" replaced with "LSPBEA Indonesia". The website will be displayed in Indonesian language and includes file upload functionality.

## Features
- Static website with identical visual design to the source file
- Complete branding replacement from "BUILDING ENGINEERS ASSOCIATION" to "LSPBEA Indonesia"
- Custom building-themed logo in header and footer components
- Preserve all original styling, colors, layout, and visual elements
- Indonesian language content throughout the site
- Consistent phone number display across all sections
- File upload functionality with blob storage
- Domain display in header and footer sections
- Clean interface with no external AI service attributions or branding
